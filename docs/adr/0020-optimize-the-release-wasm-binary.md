# ADR 0020: Optimize the release wasm binary

- Status: Accepted
- Deciders: arisu6804

## Context

The CarbonMint smart contract needs a clear, documented approach to "optimize the release wasm binary" so the codebase stays consistent and auditable.

## Decision

We optimize the release wasm binary as the standard for this contract, in line with Soroban best practices.

## Consequences

Improves clarity, testability, and maintainability, and gives future contributors a recorded rationale to build on.
