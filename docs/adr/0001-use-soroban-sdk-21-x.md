# ADR 0001: Use soroban-sdk 21.x

- Status: Accepted
- Deciders: arisu6804

## Context

The CarbonMint smart contract needs a clear, documented approach to "use soroban-sdk 21.x" so the codebase stays consistent and auditable.

## Decision

We use soroban-sdk 21.x as the standard for this contract, in line with Soroban best practices.

## Consequences

Improves clarity, testability, and maintainability, and gives future contributors a recorded rationale to build on.
