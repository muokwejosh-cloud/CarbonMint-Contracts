# ADR 0015: Embed contractmeta metadata

- Status: Accepted
- Deciders: arisu6804

## Context

The CarbonMint smart contract needs a clear, documented approach to "embed contractmeta metadata" so the codebase stays consistent and auditable.

## Decision

We embed contractmeta metadata as the standard for this contract, in line with Soroban best practices.

## Consequences

Improves clarity, testability, and maintainability, and gives future contributors a recorded rationale to build on.
