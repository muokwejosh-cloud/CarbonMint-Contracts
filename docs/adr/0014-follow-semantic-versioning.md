# ADR 0014: Follow semantic versioning

- Status: Accepted
- Deciders: arisu6804

## Context

The CarbonMint smart contract needs a clear, documented approach to "follow semantic versioning" so the codebase stays consistent and auditable.

## Decision

We follow semantic versioning as the standard for this contract, in line with Soroban best practices.

## Consequences

Improves clarity, testability, and maintainability, and gives future contributors a recorded rationale to build on.
